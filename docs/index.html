Dual API Architecture

Flexible integration options for any application

HTTP API

RESTful interface for traditional web applications and services that need standard HTTP communication. 

curl -X POST http://localhost:8080/transactions \
  -H "Content-Type: application/json" \
  -d '{"addresses": ["9xQeWvG816bUx9EPjHmaT23yvVM2ZWbrrpZb9PusVFin"]}'

gRPC API

High-performance binary protocol ideal for microservices and applications requiring low latency. 

service TransactionService {
  rpc GetTransactions(GetTransactionsRequest) returns (GetTransactionsResponse) {}
}

message GetTransactionsRequest {
  repeated string addresses = 1;
}

message GetTransactionsResponse {
  repeated Transaction transactions = 1;
}
